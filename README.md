# Pokedex ⚡

Bem-vindo à minha Pokedex! Este é um projeto que desenvolvi para explorar e exibir informações sobre Pokémons. 🧙🏼‍♂️

## 🚀 Como Usar

1. Clone este repositório:
   ```bash
   git clone https://github.com/angeloneves19/pok-dex.git
2.Navegue até o diretório do projeto:

3.No seu git-bash
cd pokedex
Instale as dependências:

4.No seu git-bash
npm install
Inicie a aplicação:

5.No seu git-bash
npm start
Agora você está pronto para explorar a Pokedex! 🎉

Inspirado por Leo Ujo 🌟
Este projeto foi inspirado por um tutorial incrível do Leo Ujo. Segui o passo a passo guiado no vídeo Parte 1 e Parte 2 para me aperfeiçoar em APIs e manipulação de dados em React. Agradeço ao Leo Ujo pela excelente explicação e por compartilhar seu conhecimento!

Funcionalidades 🌟
Lista de Pokémon: Explore uma lista com 107 Pokémon diferentes, com suas informações básicas.
Detalhes do Pokémon: Clique em um Pokémon para ver detalhes adicionais, como altura, peso, tipo, variações e ataques.
Pesquisa: Utilize a barra de pesquisa para encontrar seu Pokémon favorito rapidamente.
Tecnologias Utilizadas 🛠️
ReactJS
Material-UI
PokeAPI
